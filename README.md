# MoodBoard

An engaging daily visual inspiration app where users create and share beautiful mood boards from curated images. Each day presents a new theme (e.g., 'Serenity', 'Urban Energy', 'Nature's Power') and users drag-and-drop images to create their perfect mood board. Features voting, daily challenges, and social sharing.

Key Features:
- Drag & drop interface
- Daily themes and challenges
- Social sharing and likes
- Personal gallery of created boards
- Responsive design

Tech Stack:
- HTML5, CSS3, JavaScript
- Drag & Drop API
- LocalStorage for persistence
- CSS Grid & Flexbox
